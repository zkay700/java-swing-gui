# Java Swing GUI Basics

A collection of beginner-to-intermediate Java Swing programs demonstrating core GUI programming concepts — event handling, graphics drawing, keyboard/mouse input, and component interaction.

## Programs Included

| Program | Description |
|---|---|
| `ButtonClickCounter` | Tracks and displays the number of button clicks |
| `CardDrawing` | Renders a playing card using Java2D graphics |
| `ChangeShapeOnKey` | Changes the displayed shape based on keyboard input |
| `ClickToChangeColor` | Changes a shape's color on mouse click |
| `DrawLineBetweenClicks` | Draws a line connecting two clicked points |
| `DrawRectangleOnClick` | Draws a rectangle wherever the user clicks |
| `HouseDrawing` | Renders a house scene using basic geometric shapes |
| `KeyPressDisplay` | Displays the last key pressed on screen |
| `RandomColorCircles` | Generates randomly colored circles on the canvas |
| `RandomColoredSquaresOnKeyPress` | Spawns colored squares on each key press |
| `RightClickOvalCounter` | Counts and displays right-click ovals drawn |
| `SmileyFace` | Draws a smiley face using Java2D |
| `ToggleCircleFill` | Toggles a circle between filled and outlined on click |
| `TrafficLight` | Simulates a clickable traffic light sequence |
| `TreeDrawing` | Draws a tree using recursive or geometric shapes |
| `TriangleDrawer` | Draws triangles based on user interaction |

## Tech Stack

- **Language:** Java
- **GUI Framework:** Java Swing / AWT
- **IDE:** VS Code (with Java Extension Pack)

## How to Run

1. Open the project folder in **VS Code**
2. Make sure you have the **Java Extension Pack** installed
3. Open any `.java` file
4. Click the **Run** button (▶) in the top right, or right-click → *Run Java*

Each file is self-contained and runs independently.

## Requirements

- Java JDK 11 or higher
- VS Code with [Java Extension Pack](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-pack)
